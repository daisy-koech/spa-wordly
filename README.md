# Wordly. A Dictionary Web App
A simple dictionary that fetches word definitions, pronunciations and synonyms using a dictionary API.

## Core Features
- An input field for searching any English word
  - Displays definition
  - Shows pronunciation when available
  - Shows synonyms when available
  - Handles missing data and invalid word searches

## Built using:
- HTML to structure the search form and the ouput section
- CSS to style for readability and ease of use
- Javascript to fetch data and for DOM manipulation
- A free dictionary API

## How to set up
- clone the repository and open the project in your code editor.

## How to run
- Use Live Server VS Code extension to view the file on the browser.
- Open https://spa-wordly-delta.vercel.app/ on vercel.

## How to use
- Type an English word in the input field
- Click search or press Enter
- View definition, pronunciation (when available) and synonyms (when available)
- Invalid word search and an empty input shows an error message
